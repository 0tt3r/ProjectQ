revkit
======

This library integrates RevKit into ProjectQ to allow some automatic synthesis
routines for reversible logic.  The library adds the following operations that
can be used to construct quantum circuits:

- :class:`~projectq.libs.revkit.PermutationOracle`: Synthesizes a reversible circuit for a permutation
